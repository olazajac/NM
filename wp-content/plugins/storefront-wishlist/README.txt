== Description ==

= Storefront Wishlist =

Let's your customers save products to wishlist to buy later.

* Shop and category add to wishlist heart colors
* My Wishlist heart color.
* Single product add to wishlist heart color.
* Add to wishlist heart positions for both archives and single product pages.
* My wishlist heart button can be position Left/Right.

Storefront wishlist needs [Storefront Pro](https://www.pootlepress.com/shop/storefront-pro/ "Get Storefront Pro from PootlePress") to work.

== Installation ==

Installing "Storefront Wishlist” can be done either by searching for "Storefront Wishlist" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Make sure you have Storefront Pro installed.
2. Download the plugin via WordPress.org.
3. Upload the ZIP file through the "Plugins > Add New > Upload" screen in your WordPress dashboard.
4. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How does it work? =

After activating the plugin, it will show Add to wishlist heart icon for every product and My wishlist heart icon on bottom left.

= How do I change heart color and position? =

It can be changed in Customizer settings in Storefront Wishlist section.
